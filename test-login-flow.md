## 3. 忘记密码流程测试

### 测试步骤
1. 点击"忘记密码"
2. 弹窗从页面正中间弹出，带半透明遮罩
3. 输入手机号
4. 输入图片验证码（可点击刷新）
5. 点击"获取验证码"：
   - [x] 开始60秒倒计时
   - [x] 按钮变灰不可点击
6. 输入短信验证码（4位数）
7. 设置新密码（8-16位，2种及以上组合）
8. 确认密码
9. 点击"保存"

### 密码复杂度校验
- 长度：8-16位
- 类型：字母、数字、符号至少2种
- 两次密码必须一致

## 4. 引导设置登录密码弹窗测试

### 触发条件
- 账号密码登录成功 或 微信扫码登录绑定账号成功
- 且未设置登录密码

### 特点
- **不可关闭**：右上角无关闭按钮
- **强制设置**：必须设置密码才能访问系统
- 点击遮罩层不会关闭弹窗

### 测试步骤
1. 弹窗显示"您还未设置登录密码"
2. 输入设置密码（带眼睛图标可切换显示）
3. 输入确认密码（带眼睛图标）
4. 密码复杂度校验
5. 点击"保存"后跳转工作台

## 6. 后端API验收

### 已实现接口
- `GET /api/security-config` - 获取安全配置
- `GET /api/verify-code/:phone` - 获取图片验证码
- `POST /api/login` - 登录
- `POST /api/unlock-account` - 解锁账户

### 安全配置（可配置）
- `requireVerifyCodeAfterFailures`: 3 - 失败N次后需要验证码
- `lockThreshold`: 10 - 锁定阈值
- `lockDurationMs`: 1800000 - 锁定时长（30分钟）
- `verifyCodeExpiryMs`: 300000 - 验证码有效期（5分钟）
